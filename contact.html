What is programming ?

Programming is the process of taking an algorithm and encoding it into a notation, a programming language, so that it can be executed by a computer.In simple words “Programming is how you get computers to solve problems.”

Programming language :

Artificial language that controls the behaviour of computer.

Defined through the use of syntactic and semantic rules.

Used to express algorithms precisely.
